# Parsing_CAT_companies
Сбор данных компаний, использующих CAT систему.
При использовании парсинга компаний, имеющих cat систему и подходящих под критерий -  выручка более 100 млн. руб,
использовал открытые источники информации - сайты Superjob и Rusprofile.
На Superjob делал парсинг страниц HTML, что менее надежно, но тоже работает. Искал вакансии по клюевым словам, перечисленных в файле superjob_parser.py
На rusprofile сайте отфильтровал компании по значению выручки более 100 млн.руб.
Всего нашлось 28 компаний. И у всех одинаковое значение cat_product-Trados/Smartcat.
Хотел изначально использовать hh.ru для парсинга вакансий, но так и не дождался от них api ключа.
Затем обработал данные на пропуски и дубликаты. Их не обнаружилось.
В качестве LLM взял локальную модель Ollama gemma 2b для поиска использования либо CAT либо не CAT системы.
Сделал анализ сайтов.
Получил неожиданный результат с уровнем уверенности(cat_confidence) =  0 во всех 28 - и случаях.
Вывод - ни одна компания из списка компаний, потенциально использующих CAT системы, на основании найденных вакансий по ключевым словам,
на самом деле CAT не использует.
